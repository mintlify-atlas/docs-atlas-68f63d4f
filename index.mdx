SlideMenu brings native-feeling, gesture-driven side menus to SwiftUI apps. Drop it into any view hierarchy, customize every aspect through environment modifiers, and let `SlideMenuButton` wire itself up automatically — no bindings or callbacks required.

## Key features

<CardGroup cols={2}>
  <Card title="Edge-swipe gestures" icon="hand-pointer">
    Horizontal drag from the screen edge opens the menu. Axis locking ensures it works seamlessly alongside `ScrollView`, `List`, and `Form`.
  </Card>
  <Card title="Left and right support" icon="arrows-left-right">
    Choose which edge the menu slides from using `SlideMenuDirection.left` or `.right`.
  </Card>
  <Card title="Environment-based API" icon="leaf">
    All customization flows through SwiftUI's environment — apply modifiers anywhere in the ancestor chain, no prop drilling needed.
  </Card>
  <Card title="Zero dependencies" icon="circle-check">
    Pure Swift, no third-party packages. Distributed as a Swift Package with iOS 15+ support.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Add the package">
    In Xcode, go to **File → Add Package Dependencies** and enter the repository URL, or add it to your `Package.swift`:

    ```swift Package.swift
    dependencies: [
        .package(url: "https://github.com/matteozappia/SlideMenu.git", branch: "main")
    ]
    ```
  </Step>
  <Step title="Import and use">
    Import `SlideMenu` and wrap your root view:

    ```swift ContentView.swift
    import SlideMenu

    SlideMenu {
        Text("Main Content")
    } menu: {
        Text("Side Menu")
    }
    ```
  </Step>
  <Step title="Customize appearance">
    Apply modifiers to control the look and feel:

    ```swift
    SlideMenu { ... } menu: { ... }
        .slideMenuWidth(300)
        .slideMenuBackground(.ultraThinMaterial)
        .slideMenuAnimation(.bouncy(duration: 0.5, extraBounce: 0.15))
    ```
  </Step>
</Steps>
